Website Loading Type Analyzer (Limited Demo)

**Please Note:** This tool is a *very limited* demonstration. Due to browser security restrictions (Same-Origin Policy), it cannot accurately determine a website's "loading type" (JS or HTML based rendering) for *external* websites. It can only show the initial HTML response, which might not reflect the final rendered content if JavaScript is heavily used. For accurate analysis, server-side tools or headless browsers are typically required.
